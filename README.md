# Adversarial Iterative Constraining for Reducing AI-Detectability

- In the `CSV` folder, you find the base prompts and the search and analysis results.
- The `experiment.py` file runs the beam search procedure using a Modal app.
- Same thing for `coherence.py` for the coherence judging and `qwen_baseline_experiment.py`.
- In `Analysis_Plots.ipynb`, we created all the plots for our results.
- In `Test_Detector_Parallel_Corpus.ipynb`, we validated the used AI text detector on a dataset.

Read `report.pdf` for all infos about the project.
